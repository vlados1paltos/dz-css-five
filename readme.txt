Теоретическая часть:
    
    1. Объясните, что называется "потоком"(нормальным потоком документа), 
    и как элементы располагаются на странице по умолчанию.

    Ответ:
        Нормальный поток - это порядок, в котором браузер размещает элементы на странице 
        без применения дополнительных стилей(по умолчанию):
        1) Блочные элементы(например div, p) занимают всю ширину и располагаются друг за другом.
        2) Строчные элементы(например span, a) выстраиваются в одну линию и обтекают друг друга.

    2.  Опишите явления схлопывания отступов и выпадения отступов, 
        приведите примеры, когда эти эффекты могут возникать, и как их можно избежать.

    Ответ:
        1) Схлопывание отступов(margin collapsing):
        Если два блочных элемента имеют вертикальные отступы
        (например margin-bottom: 100px; у первого и margin-top: 100px; у второго),
        расстояние между ними будет равно наибольшему значению(100px), а не их сумме(200px).

        Пример кода:

        .block-1 {
            margin-bottom: 100px;
        }

        .block-2 {
            margin-top: 50px;
        }

        Расстояние между элементами будет 100px, а не 150px.

        Как этого можно избежать:
         - использовать padding вместо margin;
         - добавить border родителю;
         - использовать display: flex;
         - использовать overflow: hidden;
         - создать новый контекст форматирования;

        2) Выпадение отступов:
        Если у блока есть отступы, они могут "выпадать" за пределы
        родительского элемента, если у родителя нет border или padding.

        Пример кода:

        .parent {
            background-color: gray;
        }

        .child {
            margin-top: 50px;
        }

        В результате отступ будет применён не только к дочернему элементу,
        но и родитель визуально сместится вниз.

        Как этого можно избежать:
        - добавить родителю padding;
        - добавить border;
        - использовать overflow: hidden;
        - использовать display: flex;
        
    3. Объясните, что такое свойство overflow и как оно работает.
       Перечислите его основные значения (visible, hidden, scroll, auto)
       и опишите, в каких случаях их использовать.

    Ответ:
       Свойство overflow управляет поведением контента, который не помещается в заданные размеры блока:
       * visible - контент виден за пределами блока(по умолчанию).
         Подходит для декоративных элементов и случаев, когда обрезка не требуется.

       * hidden - контент обрезается границами блока.
         Часто применяется для обрезки изображений, создания карточек и устранения выпадения margin.

       * scroll - добавляются полосы прокрутки по обеим осям.
         Используется, когда полосы прокрутки должны отображаться всегда, даже если контент помещается внутри блока.

       * auto - полосы прокрутки появляются только при необходимости.
         Используется, когда полосы прокрутки должны появляться только при необходимости.

       Для текста: text-overflow: ellipsis добавляет многоточие(...)
       к обрезанному тексту(требует overflow: hidden и white-space: nowrap)

    4. Назовите способы выйти из нормального потока 
    и кратко опишите каждое свойство - position: relative, position: absolute,
    position: fixed, position: sticky, float.

    Ответ:
        Способы выйти из нормального потока:
        1) position: relative
        2) position: absolute
        3) position: fixed
        4) position: sticky
        5) float

        Описание свойств:
        * position: relative - элемент сдвигается относительно своего обычного положения,
        место в потоке сохраняется. Пример: top: 10px; сдвинет элемент вниз на 10px.

        * position: absolute - элемент убирается из потока и 
        позиционируется относительно ближайшего предка с position: relative, absolute или fixed
        (или html, если такого нет)

        * position: fixed - элемент фиксируется относительно окна браузера и не движется при прокрутке.

        * position: sticky - элемент ведёт себя как relative, пока не достигнет заданной точки
        (например top: 10px), после чего "прилипает" как fixed, пока не выйдет за пределы родителя.

        * float - это свойство позволяет элементам "плавать" слева(left) или справа(right),
        а текст и строчные элементы обтекают их.
